hubot-thecat
============

A hubot script to interact with [The Cat API](http://thecatapi.com/).

## Dependencies
* "cheerio": "~0.13.1"

## Configuration
* None

## Commands
* hubot cat me <category> - Receive a cat
* hubot cat bomb N - get N cats
* hubot show me cat's categories - show cat's active categories.
